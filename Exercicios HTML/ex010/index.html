Esse link é seu ou dos outros?

"Rel next" indica que o link é para a próxima parte do documento atual, enquanto "rel prev" é link para parte anterior do documento.

"Rel external" indica que é um link para outro site, que não faz parte do site atual.

Existe também o rel author que indica que é um link para o site do autor do artigo atual.
